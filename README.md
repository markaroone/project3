# Pokedex - Pokemon Application

This is a simple application for searching and viewing pokemons and their stats.

## Description

This application was made using React.js for the frontend and uses poke api for the data.

## Getting Started

### Executing program

* npm install
* npm start
```
npm install
npm start
```
